# Project Overview
Aerthos is a faithful recreation of Advanced Dungeons & Dragons 1st Edition as a single-player text adventure game. It features authentic mechanics including THAC0 combat, Vancian magic, and specific class/race abilities.

## Technical Stack
- **Language**: Python 3.10+
- **Dependencies**: Python Standard Library only (no external dependencies)
- **Interface**: Command-line interface (CLI) with natural language parser

## Key Features
- **Core Rules**: THAC0, descending AC, 5-category saving throws, Thief skills.
- **Classes**: Fighter, Cleric, Magic-User, Thief.
- **Races**: Human, Elf, Dwarf, Halfling.
- **Game World**: "The Abandoned Mine" starter dungeon (10 rooms).
- **Systems**: Auto-mapping, Save/Load checkpoints, Time tracking, Resource management.

## Directory Structure
- `aerthos/`: Core game logic and source code.
- `docs/`: Documentation.
- `tests/`: Automated tests.
- `web_ui/`: (Likely experimental or planned web interface).
- `main.py`: Entry point.
